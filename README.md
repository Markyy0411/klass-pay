# KlassPay 💸

[![Language](https://img.shields.io/badge/Language-TypeScript-blue.svg)](https://www.typescriptlang.org/)
[![Framework](https://img.shields.io/badge/Framework-React-61DAFB.svg)](https://react.dev/)
[![Blockchain](https://img.shields.io/badge/Blockchain-Stellar_Soroban-black.svg)](https://stellar.org/soroban)

KlassPay is a premium, gasless split-payment engine designed specifically for university students. It simplifies the process of collecting funds for group projects, class events, and shared expenses, eliminating the social friction of chasing peers for payments.

## 🛠️ Core Competencies Demonstrated

*   **Smart Contract Development:** Built and deployed robust Soroban smart contracts on the Stellar network.
*   **Blockchain Integration:** Implemented gasless transaction mechanics using `FeeBumpTransaction` for enhanced user experience.
*   **Full-Stack Development:** Orchestrated end-to-end integration between React frontend, TypeScript, and Rust-based blockchain backends.
*   **UX/UI Optimization:** Focused on frictionless user onboarding, including real-time progress tracking, toast notifications, and responsive design.
*   **Financial Technology:** Developed automated settlement logic and integrated local fiat offramping solutions (GCash).

## 📂 Project Structure & Features

*   **Instant Bill Creation:** One-click deployment of custom payment contracts.
*   **Gasless Transactions:** Background fee sponsorship ensures users pay zero gas fees, removing a critical barrier for students.
*   **Transparent Tracking:** Real-time visibility into funding progress and contributor addresses.
*   **Automated Offramping:** Direct withdrawal paths to local fiat services like GCash.
*   **Data-Driven Iteration:** Extensively iterated based on user feedback (100+ mainnet users), including CSV export functionality and dark/light mode accessibility.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Markyy0411/klass-pay.git
   cd klass-pay/frontend
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```
